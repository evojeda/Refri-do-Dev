🧃 Refri do Dev

Projeto de interface web desenvolvido com HTML, CSS e JavaScript, simulando uma experiência simples e interativa de navegação por telas de um produto fictício.

🔗 Acesse o projeto online:
https://evojeda.github.io/Refri-do-Dev/

📌 Sobre o projeto

O Refri do Dev é uma página web com três telas que apresentam um conceito criativo de produto voltado ao universo da tecnologia.

O objetivo do projeto foi praticar:
estruturação de páginas com HTML;
estilização com CSS;
organização visual de interface;
navegação entre telas.
